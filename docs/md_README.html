Command Line Arguments

The program takes in 5 arguments with the first being a directory path (string) and the remaining being pixel dimensions (integer). Argument Format:

"PATH_TO_IMAGE_FOLDER" "WALL_ONE_WIDTH" "WALL_ONE_HEIGHT" "WALL_TWO_WIDTH" "WALL_TWO_HEIGHT"

Algorithms Implemented

Our algorithm uses a "spiral" like strategy to place images, filling the outside edges first before working towards the middle.

  • Expensive First - Inserts the most expensive images first onto the wall
  • Smallest First - Inserts the smallest images onto the wall first
  • Heuristic - Uses weightings and other algorithmic evaluations to prioritize the insertion of certain images before others
