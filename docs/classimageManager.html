Detailed Description

Adapter class for CIMG library.

The imageManager is the class which will abstract most of the individual functionalities of the Cimg library in a more user-friendly readable fashion.

For the purposes of this project, each "wall" will have its own image manager.

◆ removeLeak()

void imageManager::removeLeak ( )

special deletion of pictures which plugs a memory leak this funciton is needed because of the way that imageManager is implemented. when chooseFirst is called on the first wall, it returns a vector of picture*, and that is the failed_Placements vector. However, when the second wall has chooseFirst called, then the failed_Placements are never deleted if there are pictures that could not fit on the second wall. So, this function is called to delete those failed placements after the second wall has had chooseFirst() called.
